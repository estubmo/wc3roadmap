The choice determines the SPDX identifier on every source file, the LICENSE wording, and the package.json license field. RESEARCH's working default across Phase 1 source files is GPL-3.0-or-later (the more permissive, forward-compatible choice, and the value RESEARCH attributes to wc3v). This checkpoint exists so a human confirms before the binding LICENSE is committed.   </context>

      <pros>Forward-compatible with future GPL versions; matches RESEARCH's reading of wc3v; already the default in Phase 1 source headers (no sweep needed)</pros>
      <cons>Allows relicensing under a later GPL the author has not read</cons>

    <option id="only">
      <name>GPL-3.0-only</name>
      <pros>Maximum certainty of matching wc3v if wc3v turns out to be -only; no implicit future-version grant</pros>
      <cons>Requires sweeping existing "-or-later" SPDX headers to "-only"; less forward-compatible</cons>
    </option>
  </options>
