<objective>
Add the quiz content model to the node frontmatter schema: a `quiz` array of 3–5 multiple-choice questions, each question carrying options (exactly one correct) and a REQUIRED `explanation` (the QUIZ-03 structural guardrail). The schema lands in BOTH `src/schemas/node.ts` and `content-collections.ts` in the same wave (D-01 parallel-schema sync), so build-time and runtime validation never drift.

Purpose: Quiz content is authored co-located in each node's MDX (ADR 002) and validated by the existing content-collections Zod pipeline. The required `explanation` forces authors to justify recall depth — a structural mechanism for QUIZ-03, not a checklist.
Output: `QuizSchema`/`QuizQuestionSchema`/`QuizOptionSchema` + types, an optional `quiz` field on `NodeFrontmatterSchema`, a mirrored schema + count guard in `content-collections.ts`, and colocated tests.
</objective>
